App Track: Technical Breakdown

Frontend React

The frontend of AppTrack is built using React, providing a clean and responsive interface for managing job applications. It features a searchable and filterable table that displays all applications, with fields like job title, company name, application date, status, and city. Users can add, edit, and delete jobs directly from the UI, with dynamic state updates and modal dialogs for a smoother UX.

Additionally, there's a feature that allows users to input a LinkedIn job offer URL, which triggers a backend process to auto-fill job information using AI and scraping. This enhances usability by reducing manual entry and helps users focus on managing their job search effectively.

Java Spring Boot Backend

import jakarta.persistence.*; 

@Entity
@Table(name = "jobs")
public class Job {
    
    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    private String jobTitle;
    private String companyName;
    private String status;
    private String dateApplied;
    private String city;
    @Column(columnDefinition = "TEXT")
    @Lob
    private String description;
}

The backend is powered by Java Spring Boot, exposing a robust REST API for full CRUD operations on job applications. The API handles endpoints to create, retrieve, update, and delete job entries, and integrates seamlessly with the React frontend. Data is managed using standard Java entity models and persisted via a lightweight in-memory database (or a traditional RDBMS, depending on config).

Spring Boot also includes CORS configuration to allow communication with the React frontend. The backend ensures data integrity and handles JSON payloads for both manual input and AI-generated job data, acting as the core logic and persistence layer of the application.

@Configuration
public class CorsConfig {

    @Bean
    public WebMvcConfigurer corsConfigurer() {
        return new WebMvcConfigurer() {
            @Override
            public void addCorsMappings(@NonNull CorsRegistry registry) {
                registry.addMapping("/**")
                        .allowedOrigins("http://localhost:5173") // Autorise le frontend
                        .allowedMethods("GET", "POST", "PUT", "DELETE", "OPTIONS") // Autorise les méthodes HTTP
                        .allowedHeaders("*") // Autorise tous les en-têtes
                        .allowCredentials(true); // Autorise les cookies si nécessaire
            }
        };
    }
}

Python AI Scraper

The AI-powered scraping functionality is handled by a lightweight Python FastAPI server, which takes in a LinkedIn job URL and extracts relevant job details (title, company, city, description, etc.). This data is returned as structured JSON, ready to be consumed by the Java backend or directly used in the React interface.
