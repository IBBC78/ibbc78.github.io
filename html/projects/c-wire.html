C-Wire is a command-line tool developed in C and Shell, designed to analyze large CSV files that simulate an energy distribution network. The project was carried out as a team effort, with a focus on both technical performance and collaborative workflow. The tool’s main objective is to process and classify electricity consumption data associated with high-voltage and low-voltage substations, providing clear and actionable output for further analysis.

The frontend of C-Wire is a straightforward terminal interface, where users can execute various commands to analyze specific aspects of the dataset. The tool offers multiple functionalities, such as sorting substations by capacity, generating CSV output files, and producing visual histograms using GnuPlot scripts. The output files are organized for easy interpretation, and the tool is optimized for speed and efficiency, even with large datasets.

On the backend, C-Wire leverages advanced data structures—such as AVL trees—for efficient data storage and retrieval. The Shell scripts handle data preprocessing and argument parsing, while the C modules perform the heavy computation and sorting tasks. One of the project’s key achievements was optimizing the workflow to minimize processing time: by piping data directly from Shell to C and reducing the use of temporary files, the team achieved significant performance gains. The project also includes automated generation of PDF reports containing the produced histograms, thanks to LaTeX integration.

C-Wire is a testament to effective teamwork, code optimization, and the practical application of algorithms in real-world data processing. The project showcases proficiency in both systems programming and scripting, as well as the ability to deliver clear, well-documented results.
