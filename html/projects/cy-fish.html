CY-Fish: A Terminal-Based Strategy Game

CY-Fish is a turn-based strategy game developed in C for the terminal, designed as a collaborative academic project. The game immerses players in a challenging environment where they control penguins on a melting iceberg, competing to collect as many fish as possible before the board disappears. The gameplay is based on a hexagonal grid, requiring players to anticipate their opponents' moves and adapt to the shrinking play area.

The frontend of CY-Fish is entirely text-based, using ASCII graphics to represent the board, penguins, and fish. The interface is optimized for clarity and quick decision-making, allowing players to easily visualize their positions and plan their strategies. Although visually simple, the terminal interface provides a retro gaming experience and ensures compatibility across different systems.

On the backend, the project features robust logic for managing game turns, movement validation, and dynamic board updates as the iceberg melts. The code is structured to ensure fair play, with clear rules for movement and fish collection. Special attention was given to handling edge cases, such as blocked paths or simultaneous moves. The project also includes a modular design, making it easy to extend or adapt for new game modes or features in the future.

CY-Fish demonstrates strong skills in algorithmic problem-solving, memory management, and collaborative software development in C. The project highlights the ability to design engaging gameplay mechanics while maintaining code clarity and efficiency.
