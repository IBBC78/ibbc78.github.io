Marvel Travel

Current Project

Marvel Travel is a full-stack web application developed as part of an academic project. It simulates a complete travel booking experience, from browsing destinations to managing reservations and user accounts.

The platform was built using PHP for the backend, with HTML, CSS, and JavaScript on the frontend. It includes session-based user authentication, dynamic content loading, and simulated payment flows. One of the key goals of the project was to design a modular, easy-to-extend architecture, while ensuring a smooth user experience.

This project helped me strengthen my skills in backend logic, interface design, and database interaction — all within the context of a realistic, user-focused scenario.
